Установка Rabbit Hole:

1. Загрузка приложения

Скачайте приложение Rabbit Hole по ссылке:

⬇️ Скачать Rabbit Hole

Установите приложение "Rabbit Hole".

2. Откройте приложение

Откройте приложение, нажмите кнопку "нажмите, чтобы добавить настройки VPN".

3. Разрешите доступ к буферу обмена

Разрешите приложению доступ к буферу обмена, нажав "Разрешить вставку".

4. Вставка из буфера

Скопируйте вашу ссылку для подключения (нажмите кнопку "⚡️ Подключиться!" в боте)

Нажмите "Вставка из буфера".

5. Выбирите локацию(страну)

Нажмите "Подключиться".
